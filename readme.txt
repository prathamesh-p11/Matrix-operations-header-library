BrainCorp Matrix operations take home assignment

Description
A portable linear algebra header-only library transpose and multiplication of MxN matrices.

Dependencies
	g++
	C++11 or higher
	Linux
	MinGW (only for windows)
    
Executing program:
On linux:
	Open terminal
	Enter command : make to compile and build test file
	Enter ./mat to run the program

On windows:
	Open terminal
	Enter command : MinGW32-make to compile and build test file
	Enter mat.exe to run the program
